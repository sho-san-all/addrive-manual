> 📍 この機能は **サイドバー > ダッシュボード** にあります

## 概要

ダッシュボードは、当月の広告運用状況を一覧で把握するための画面です。案件別・媒体別の実績をグラフと数値で確認できます。

## 画面の見方

ダッシュボードは上から順に以下のセクションで構成されています。

| セクション | 内容 |
|---|---|
| KPIサマリー | 月間予算・消化額・達成率をカード形式で表示 |
| 案件別グラフ | 案件ごとの予算消化状況を棒グラフで比較 |
| 媒体別内訳 | Google・Meta等の媒体ごとの配信実績 |
| アラート一覧 | 予算超過・入稿ミス等の要注意項目 |

## 操作手順

### 期間を切り替える

1. 画面右上の **期間選択** ドロップダウンをクリック
2. 「今月」「先月」「カスタム」から選択
3. カスタムを選んだ場合は開始日・終了日をカレンダーで指定
4. **適用** ボタンをクリックすると画面が更新される

### データをエクスポートする

1. 画面右上の **エクスポート** ボタンをクリック
2. 形式（CSV / Excel）を選択
3. ダウンロードが自動で始まる

## 注意事項 / よくあるつまずき

- **データが更新されていない** — データは1時間ごとに自動更新されます。最新にしたい場合は右上の **更新** アイコンをクリックしてください。
- **グラフが表示されない** — 選択した期間に実績データがない場合は空で表示されます。期間を変更してお試しください。
